# React E-Commerce Admin Dashboard

A simple React-based admin dashboard with authentication and full CRUD functionality for managing products.

## Features
- User login and registration
- Protected routes
- Axios interceptor for authentication
- Create, read, update, and delete products
- Responsive design
- Delete confirmation

## Tech Stack
- React
- Vite
- React Router
- Axios
- CSS

## Notes
This project uses a mock API, so created products may not persist after refresh.

## Getting Started

```bash
npm install
npm run dev
